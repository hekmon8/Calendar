# Calendar
  - Calendar component support select date.
  
### Installation
```sh
$ cd Example
$ pod install
```

### Tech
Calendar uses technology:

* [Swift] - Language to write application for iOS ..etc
* [Cocoapods] - The Dependency Manager for iOS & Mac projects.
